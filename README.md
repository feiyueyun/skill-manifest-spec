# Skill Manifest Specification

The open standard for describing AI Agent skill services.

> 🚧 This specification is under active development. RFC process coming soon.

## Overview

Skill Manifest (`skill.json`) is a JSON-based format for describing AI Agent skills — their capabilities, requirements, protocols, and metadata. It enables agents to discover, evaluate, and invoke skills across secure mesh networks.

- **Current Version**: Draft
